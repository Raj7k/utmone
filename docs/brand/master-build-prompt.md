## 🎨 UI/UX REQUIREMENTS

### Typography

* Use Inter
* H1: 64–72px / bold
* H2: 36–40px
* H3: 24–28px
* Body: 16–18px
* Caption: 12–14px

### Color Palette

* Background: #FAFAF9
* Text: #18181B
* Muted text: #71717A
* Accent: #6366F1
* Success: #10B981
* Error: #EF4444

### Spacing System

* 128px section padding
* 96px block margin
* 64px between cards
* 32px between content
* 16px inner padding

### Component Philosophy

* Soft shadows
* Light borders (rgba(0,0,0,0.08))
* Purposeful animations (180–240ms)
* Never more than one primary CTA

### Layout

* 12-column grid
* Max width: 1200–1320px
* Center-aligned hero
* Clean modular cards
